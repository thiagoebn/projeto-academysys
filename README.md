# AcademySys

Sistema acadêmico desenvolvido em Python com interface gráfica.

## Funcionalidades
- Cadastro de usuários (professor e aluno)
- Sistema de login
- Cadastro de alunos
- Lançamento e consulta de notas

## Tecnologias
- Python
- Tkinter
- JSON

```bash
python nome_do_arquivo.py
